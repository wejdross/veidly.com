= Veidly - Social Event Platform

== Overview

Veidly is a privacy-focused social event platform that allows users to create, discover, and join local events while maintaining control over their personal information.

== Quick Start

=== Development

[source,bash]
----
# Backend
cd backend
go run .

# Frontend  
cd frontend
npm install
npm run dev
----

[source,bash]
